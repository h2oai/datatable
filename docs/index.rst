
|logo|Datatable
================
.. |logo| image:: _static/py_datatable_logo.png
          :width: 80
          :class: no-scaled-link

**Datatable** is a python library for manipulating tabular data.
It supports out-of-memory datasets, multi-threaded data processing,
and flexible API.
