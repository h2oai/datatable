    Get categories for categorical data.

    Parameters
    ----------
    cols: FExpr
        Input categorical data.

    return: FExpr
        f-expression that returns categories for each column
        from `cols`.

    except: TypeError
        The exception is raised when one of the columns from `cols`
        has a non-categorical type.
