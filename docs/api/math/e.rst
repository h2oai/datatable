
.. xdata:: datatable.math.e
    :src: src/datatable/math.py e

    The base of the natural logarithm :math:`e`, also known as the Euler's
    number. This number is defined as the limit
    :math:`e = \lim_{n\to\infty}(1 + 1/n)^n`.

    The value is stored at `float64` precision, and is equal to
    ``2.718281828459045``.
