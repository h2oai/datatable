    Inverse trigonometric tangent of `x`.

    This function satisfies the property that ``tan(arctan(x)) == x``.

    See also
    --------
    - :func:`atan2(x, y)` -- two-argument inverse tangent function;
    - :func:`tan(x)` -- the trigonometric tangent function.
